# CardioScope

CardioScope is a project for analyzing and predicting cardiac data. It includes scripts for training and prediction, and organizes data into normal and abnormal categories.

## Project Structure

- `main.py`: Main script for the project.
- `predict.py`: Script for making predictions on cardiac data.
- `data/`: Contains subfolders for `abnormal` and `normal` data.
- `training/`: Contains training data divided into multiple sets (e.g., `training-a/`, `training-b/`, etc.), each with `.hea` files.

## Usage

1. **Environment Setup**
   - Ensure you have Python and required dependencies installed.
   - (Optional) Use a virtual environment or Conda environment for package management.

2. **Prediction**
   - Run predictions using:
     ```bash
     python predict.py
     ```

3. **Training**
   - Training scripts and instructions can be added as needed.

## Data
- The `data/` folder contains categorized data for model input.
- The `training/` folder contains multiple sets of `.hea` files for model training.
